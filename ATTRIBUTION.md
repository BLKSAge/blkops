# Attribution

## Primary Upstream: career-ops-plugin

**Repository:** https://github.com/andrew-shwetzer/career-ops-plugin
**Author:** Andrew Shwetzer
**License:** MIT

The career-ops-plugin provided the complete skill architecture BlkOps is built on:
skill definitions (evaluate, tailor-resume, scan, triage, track, apply, research,
outreach, compare, help), the 15-archetype detection and scoring system, ATS rules,
profile schema, resume HTML template, and plugin configuration format.

BlkOps adapted this work for Claude.ai Projects and Claude Code, extending it with
target lists, DoD evaluation adjustments, session handoff system, and resume pipeline.
